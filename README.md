# Calculadora
Calculador basica con interfaz intuitiva para el usuario

Pasos para la instalación y correcta prueba de esta:

### 1. Clona el repositorio

```bash
git clone https://github.com/kennerSteven/Calculadora.git
cd Calculadora
```

### 2. Instala Node.js 
Asegúrate de tener Node.js v18+ instalado. Verifica con


```bash
node -v
npm -v

```
Si no lo tienes, descargalo desde https://nodejs.org/es

### 2. Configuración del proyecto

```bash
npm install
```
### 2. Ejecutar proyecto

```bash
npm run dev
```
